# Serie Fibonacci

Este proyecto implementa un programa en Python para calcular la secuencia de Fibonacci y encontrar el valor en una posición específica.

## Uso

Para ejecutar el programa, simplemente ejecuta el archivo `main.py`. El programa solicitará al usuario que ingrese la posición en la secuencia de Fibonacci y luego calculará el valor correspondiente.

```python
python main.py

Ejemplo de código
El código en main.py utiliza la función fibonacci del archivo fibonacci.py para calcular la secuencia de Fibonacci.
